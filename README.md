# Простой Трейдинг — лендинг на Next.js

## Запуск

```bash
npm install
npm run dev
```

Сайт будет доступен по адресу `http://localhost:3000`.

## Где менять контент

- Основной текст, кнопки и список статей: `content/site-content.js`
- Структура страницы: `app/page.jsx`
- Стили и визуал: `app/globals.css`

## Файл для скачивания

PDF для печати расположен в `public/print_tradiing.pdf`.
